## 제어문
제어문은 프로그램의 수행 순서를 제어한다. 제어문의 종류에는 조건문, 반복문, 분기문이 있다.

| **제어문 종류** | **설명**                                                               |
|----------------|-------------------------------------------------------------------------|
| 조건문         | 프로그램 실행 흐름을 조건에 따라 분기시키는 역할을 한다.                |
| - `if`         | 조건이 참일 경우 특정 블록을 실행한다.                                 |
| - `else`       | `if` 조건이 거짓일 경우 특정 블록을 실행한다.                         |
| - `else if`    | 여러 조건을 순차적으로 확인하며 특정 블록을 실행한다.                  |
| - `switch`     | 특정 값에 따라 실행 블록을 선택해 실행한다.                           |
| 반복문         | 특정 작업을 반복적으로 수행하기 위해 사용한다.                          |
| - `for`        | 반복 횟수가 정해진 경우 사용하며 초기화, 조건, 증감 부분을 포함한다.    |
| - `while`      | 조건이 참인 동안 계속해서 반복을 실행한다.                            |
| - `do while`   | 조건 확인 전에 한 번 실행하고 이후 조건이 참이면 반복을 실행한다.       |
| 분기문         | 실행 흐름을 특정 위치로 이동하거나 작업을 종료하는 역할을 한다.         |
| - `break`      | 반복문 또는 `switch` 문을 종료한다.                                   |
| - `continue`   | 현재 반복을 건너뛰고 다음 반복으로 진행한다.                          |
| - `goto`       | 지정된 레이블로 실행 흐름을 이동한다.                                |
| - `return`     | 함수 실행을 종료하고 호출자에게 값을 반환한다.                        |

## 조건문

같은 프로그램도 사용자의 선택에 따라 다르게 동작할 수 있다. 예를 들어 동영상 플레이어에서 한글 자막이나 영문 자막을 선택적으로 보여주거나 아예 자막을 표시하지 않을 수 있다. 조건문은 조건에 따라 프로그램이 선택적으로 특정 문장을 수행하게 만드는 기능을 제공한다.

### (1) IF ~ ELSE
if문은 () 안에 있는 조건식을 참이면 주어진 문장을 수행하고, 거짓이면 다음 `else if`를 실행하거나 `else` 또는 아무 작업도 수행하지 않는다. 기본적인 if문의 사용 형식은 다음과 같다.

```c
if( 조건식 ){ 참일경우 실행할 문장 }else{ 거짓일 경우 실행할 문장 }
```

![img](/theory/Chapter/04/image1.png "if문 예제")

if문은 조건식의 참/거짓 여부에 따라 실행 흐름이 나누어지며, 각각의 블록이 실행된 후에는 다시 다음 명령에서 합쳐져 실행을 이어나간다. if문의 예제를 보자면 다음과 같다.

```c
#include <stdio.h>

int main() {
    int number;

    printf("숫자를 입력하세요: ");
    scanf("%d", &number);

    if (number % 2 == 0)/*조건식*/ { // 참일경우
        printf("결과 :\n");
        printf("입력한 숫자는 짝수입니다.\n");
    } else { // 거짓일경우
        printf("결과 :\n");
        printf("입력한 숫자는 홀수입니다.\n");
    }

    return 0;
}

// 출력 :
// 숫자를 입력하세요: 4
// 결과 :
// 입력한 숫자는 짝수입니다.
```

if문은 조건식이 참일 때 수행할 문장이 여러 개면, 수행할 문장들을 {} 로 묶어주어야 한다. 이처럼 {}를 묶인 문장을 **복합문** 또는 **블록**이라고 한다.

위 예제를 보자면 if문에서 조건식인 `number % 2 == 0`는 2로 나눈 나머지가 0인지 확인하고, 만약 나머지가 0 이라면(참이라면) 블록 안 문장인 `rintf("결과 :\n"); printf("입력한 숫자는 짝수입니다.\n");` 를 출력한다.

 나머지가 0이 아닌 다른 숫자라면 거짓이므로 `else` 블록 안 문장인 `printf("결과 :\n"); printf("입력한 숫자는 홀수입니다.\n");` 가 출력 한다. 이렇게 if문으로 통해 숫자가 짝수인지 홀수인지 확인할 수 있다.

### (2) ELSE IF
`if else`의 `else` 블록 안에 다른 문장 없이 또 다른 if문만 들어있을 때, `else if`로 작성 할 수 있다. **`else if`는 여러 가지 조건을 순서대로 검사한다.** `else if` 형식은 다음과 같다.

```c
#include <stdio.h>

int main() {
    int score;

    printf("점수를 입력하세요: ");
    scanf("%d", &score);

    if (score >= 90) {
        printf("A등급입니다.\n");
    } else if (score >= 80) {
        printf("B등급입니다.\n");
    } else if (score >= 70) {
        printf("C등급입니다.\n");
    } else {
        printf("F등급입니다.\n");
    }

    return 0;
}
// 출력 :
// 점수를 입력하세요: 85
// B등급입니다.
```
`else if`는 항상 if문과 함께 되며, `else` 앞에 있는 if문의 조건식이 거짓일 때만 `else if`다음의 조건식을 검사한다. `else if`에 또 다른 `else if`를 여러 번 연결할 수 있으며, 마지막에 `else`도 사용될 수 있다. 위 예제처럼 입력받은 점수로 조건식으로 통해 각 등급을 매길 수 있다.

### (3) SWITCH
switch문은 ()의 정수식의 값에 따라 여러 가지 `case`중 하나를 선택하여 문장을 수행한다. switch문의 사용 형식은 다음과 같다.

```c
switch( 정수식 ){
    case 정수값1:
        문장1;
        break;
    case 정수값2:
        문장2;
        break;
    default:
        문장n;
        break;
}
```

switch문의 사용 예시는 아래 예제와 같다.
```c
#include <stdio.h>

int main() {
    int score;

    printf("점수를 입력하세요: ");
    scanf("%d", &score);

    switch (score / 10) {
        case 10:
        case 9:
            printf("A등급입니다.\n");
            break;
        case 8:
            printf("B등급입니다.\n");
            break;
        case 7:
            printf("C등급입니다.\n");
            break;
        case 6:
            printf("D등급입니다.\n");
            break;
        default: // 값이 일치하는 case가 없으면 default를 찿는다.
            printf("F등급입니다.\n");
    }

    return 0;
}
// 출력 :
// 점수를 입력하세요: 85
// B등급입니다.
```
**`switch`에 ( )안에 써주는 수식의 값은 반드시 정수여야 한다.** `switch`의 {} 안에는 여러 개의 `case`를 사용할 수 있다. `case`을 작성할 때는, `case` 다음에 정수값, `;`, 수행할 문장을 순서대로 써준다. `case`문에서 수행할 문장이 여러 개일 때도, {}를 써줄 필요가 없다. 이미 `switch` 전체에 {}에 사용되기 때문이다.

**`case`에서 수행할 문장의 끝에는 `break`를 써준다.** `switch` 안에서 `break`를 만나면 `switch`를 탈출한다. `break`가 없으면 `switch`의 끝을 만날 때까지 나열된 모든 문장을 수행한다. 일치하는 `case`가 없을 때 수행할 문장을 `default`를 다움에 써준다. `default`도  수행할 문장 끝에도 `break`가 필요하다.

### (3-1) SWITCH와 IF ELSEIF ELSE
`switch`와 `if elseif else`중 어떤 것을 사용할지는 프로그래머가 결정 할 수 있다. 일반적으로 정수값을 비교하고 비교할 값이 2개 이상이라면 `switch`을 사용하는 것이 좋다.

반면에, 조건식이 단순한 정수값 비교가 아닌 경우에는 `if elseif else`를 사용하는 것이 좋다. 예를 들어 값을 비교해야 하거나 실수나 문자열을 비교해야 하는 경우는 `if elseif else`를 사용하는 것이 좋다.

## 반복문
**반복문은 조건이 만족하는 동안 주어진 문장을 반복해서 수행한다.** 반복문을

### (1) FOR
for문은 정해진 횟수만큼 반복 수행 할 때 주로 사용된다. for문은 세 가지 반복문중에서 가장 정형화된 반복문이다. 기본적인 for문의 사용 형식은 다음과 같다.

```c
for(초기식; 조건식; 증감식;){ 반복할 문장 }
```

**for문은 초기식, 조건식, 증감식과 반복할 문장으로 구성된다.** 반복할 문장이 여러 개일때는 { }로 묶어준다.

`for`에서는 먼저 초기식을 수행한 다음에 조건식을 검사한다. 조건식이 참이면 반복할 문장을 수행하고 나서 증감식을 수행한다. 여기까지가 반복 1회차에 해당한다. 초기식은 반복 1회차ㅇ에만 수행하고, 반복 2회차부터는 "조건식 > 반복할 문장 > 증감식 " 순으로 수행한다. 조건식이 거짓이면 더 이상 수행하지 않고 for문을 빠져 나간다. 순서도로 하자면 아래 사진과 같다.

![img](/theory/Chapter/04/image3.png "for문 예제")

아래는 for문을 이용한 예제이다.

```c
#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("현재 숫자: %d\n", i);
    }
    return 0;
}

// 출력 :
// 현재 숫자: 1
// 현재 숫자: 2
// 현재 숫자: 3
// 현재 숫자: 4
// 현재 숫자: 5
```

**for문은 어떤 문장을 N번 반복 수행하는 용도로 사용되는 경우가 많다.** N번 반복 수행하는 코드는 위와 같은 형태가 된다. 또한 N번 반복 수행하는 대신 다른 방법으로 for문을 사용할 수도 있다. i를 증가하는 대신 감소할 수도 있고, 5씩 더하거나 2씩 곱할 수도 있다. 이때도 조건식이 참인 동안 반복 수행 된다.

```c
#include <stdio.h>

int main(void) {
    int i;

    for (i = 1; i <= 10; i *= 2)/* i*=2가 루프 제어 변수 */ {
        printf("현재 숫자: %d\n", i);
    }
    return 0;
}

// 출력 :
// 현재 숫자: 1
// 현재 숫자: 2
// 현재 숫자: 4
// 현재 숫자: 8
```
또한 for의 초기식. 조건식. 증감식에서 사용되는 변수를 **루프 제어 변수**라고 한다. 루프 제어 변수의 값은 for의 반복 회차마다 변경되어야 하며, 특정 시점에는 조건식이 거짓이 되어 루프를 탈출 할 수 있다.

### (1-1) FOR 변형
**for 문에서 루프 제어 변수를 여러 개 사용할 수 있다.** for의 초기식은 한 문장이지만, 콤마 연산자를 이용해서 `i = 0;` `j = 100;`처럼 수식을 나열 할 수 있다. 증감식도 마찬가지다.

```c
#include <stdio.h>

int main() {
    for (int i = 0, j = 100; i <= 10 && j >= 90; i++, j--) {
        printf("i: %d, j: %d\n", i, j);
    }
    return 0;
}

// 출력 :
// i: 0, j: 100
// i: 1, j: 99
// ---
// i: 9, j: 91
// i: 10, j: 90
```
또한 **for문을 구성하는 초기식, 조건식, 증감식과 반복할 문장을 모두 생략할 수 있다.** 그리고 초기식, 조건식, 증감식을 생략하더라도 세미클론`;` 을 반드시 써줘야 한다.

for문에서 조건식을 생략하면 **무한 루프**가 된다. 무한 루프는 무한히 실행되므로, 프로그램이 일단 무한 루프 들어간다면 강제 종료해야 한다.
```c
#include <stdio.h>

int main() {
    for (;;) {
        printf("무한 루프 실행 중...\n");
    }
    return 0;
}

// 출력 :
// 무한 루프 실행 중...
// 무한 루프 실행 중...
// 무한 루프 실행 중...
// ---
```
for 문의 반복할 문장도 세미클론`;`으로 통해 생략이 가능하며 이로 이루어진 문장을 널 문장이라 한다.

```c
#include <stdio.h>

int main() {
    int i; 
    for (i = 0; i < 3; i++)
        ;

    printf("i = %d\n", i);

    return 0;
}

// 출력 :
// i = 3
```
### (1-2) 중첩된 FOR
for문 안에 포함된 for문을 **중첩된 for**라고 한다. 중첩된 외부 for에서 for가 `N`번, 내부 for가 `M`번 반복 수행하는 경우에 전체 반복 횟수는 `M*N`이 된다.
```c
#include <stdio.h>

int main() {
    for (int i = 2; i <= 3; i++) {
        printf("=== %d단 ===\n", i);
        for (int j = 1; j <= 9; j++) {
            printf("%d x %d = %d\n", i, j, i * j);
        }
        printf("\n");
    }
    return 0;
}

// 출력 : 
// === 2단 ===
// 2 x 1 = 2
// 2 x 2 = 4
// 2 x 3 = 6
// ---
// 3 x 7 = 21
// 3 x 8 = 24
// 3 x 9 = 27
```

외부 for문에서는 루프 변수 `i`를 2부터 시작해 3까지 증가시키며 반복한다. 이 루프는 구구단의 각 단을 처리하는 역할을 하고, `printf("=== %d단 ===\n", i);`를 사용해 현재 출력 중인 단을 나타낸다.

내부 for문은 외부 루프 변수 `i`의 값에 따라, 즉 현재 출력 중인 구구단의 단에 따라 `j`를 1부터 9까지 증가시키며 반복한다. 이 루프에서는 `printf("%d x %d = %d\n", i, j, i * j);`를 실행해 구구단의 단과 곱해지는 숫자, 그리고 그 결과를 출력한다.

### (2) WHILE
while문의 사용 형식은 다음과 같다.
```c
while( 조건식 )
    반복할 문장;
```

[ 추가 예정 ]
